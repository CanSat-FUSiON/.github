## (確認)Pull Requestの名前
- commit messageやpull requestは英語で記入．その下のdescriptionなどは言語指定なし．わかるように書く．  
- ”`:`”をディレクトリ階層の区切りとして階層を示し，そのディレクトリにおける作業内容を書く． 
例）`zero/rpios/~~.py`を編集し画像処理の二値化機能を追加した場合のPR `zero: rpios: Add binarize func`
